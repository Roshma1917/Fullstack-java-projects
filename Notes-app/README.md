# Fullstack Java Projects

Welcome to my coding playground.

This repository is a collection of projects built while exploring the world of full-stack development. Every folder here represents a challenge, a lesson learned, a bug conquered, and a step forward in becoming a better software engineer.

What started with a simple "Hello World" has grown into building REST APIs, connecting databases, designing user interfaces, and creating complete applications from backend to frontend.

Each project tells a story:

* Understanding how applications communicate through APIs.
* Learning how data flows from the database to the browser.
* Turning ideas into working software.
* Making mistakes, debugging, refactoring, and improving.
* Building confidence one project at a time.
